# ATM Banking System (Python CLI)

This is a command-line based ATM / banking system built using Python.

## Features
- PIN-based security with limited attempts
- Deposit and withdrawal functionality
- Minimum balance enforcement
- Transaction history tracking
- Simple interest calculator
- Menu-driven CLI interface

## Concepts Used
- Functions
- Loops and conditionals
- Lists and global variables
- Basic security logic
- Input validation

This project was created for practice and learning purposes.
